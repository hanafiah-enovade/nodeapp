# A Simple API app based on Node.js and Express

## API Routes

- `GET /`  
  Returns plain text: `Salam Sejahtera!`

- `GET /about`  
  Returns JSON: `{ "msg": "Aplikasi nodeJS di AIW Lab" }`

## Guide to install this app

### Clone this app
```
$ git clone https://github.com/Hanafiah-Enovade/nodeapp.git
```

### Install the app
```
$ cd nodeapp
$ npm install
```

### Run the app
```
$ node index.js
```

### Test the app
Open in your browser:
- [http://localhost:4000](http://localhost:4000)
- [http://localhost:4000/about](http://localhost:4000/about)
